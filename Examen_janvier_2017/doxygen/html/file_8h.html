Loads a List contained in a binary file.

This function initializes a list : it fills it with data read from a given file and set the current pointer to a given position. Structure of the binary data in the file is presented in saveList() documentation.

It is assumed that the received pointer l points towards a previously allocated List.

Parameters
[in,out]lthe List which is to be filled with data read from the file.
[in]fileNamethe path to the file containing the data.
void saveList ( List *  l,
char *  fileName 
)

Saves a List into a binary file.

This function creates a file and fills it with data read from a given List.

Binary data in the file is structured as follows:

  • the file starts with an int: corresponding to the index of the current ListNode in the List (where 0 corresponds to the first ListNode, 1 to the second one, ...).
  • This int value is followed by the char values, starting from the first node to the last one.

The List structure must not be modified in any way when calling this function, ie the current element should remain the same than before calling saveList.

Parameters
[in]lthe List which is to be saved into the file.
[in]fileNamethe path to the file where the List is saved.
